# セキュリティ対策

## パスワードの保護
パスワードはそのまま保存せず、bcryptでハッシュ化して保存しています。

## ログイン状態の管理
ログイン状態はセッションで管理しています。ログアウト時には、セッションの情報を削除します。

## 不正な送信への対策
フォームにはCSRFトークンを使用し、外部から勝手に送信されることを防いでいます。

## SQLインジェクション対策
SQLへ値を直接つなげず、プレースホルダを使用しています。

## 画像の保護
アップロードできる画像形式を制限しています。
また、ログイン中のユーザーが所有する画像だけを表示し、他のユーザーの画像を閲覧できないようにしています。
