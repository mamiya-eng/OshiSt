# OshiSt

## アプリ概要
OshiStは、推し活グッズの購入品・予約品・支出をまとめて管理するWebアプリです。
以前、購入済みの商品を忘れて同じものを買ってしまった経験から、このアプリを作りました。

## 主な機能
- 購入品の登録・編集・削除
- 画像の登録
- 入力途中での下書き保存
- 同じ商品がないか確認する、かぶり検知
- 月ごとの予算管理
- 購入金額の振り返り

## 使用技術
- Python
- Flask
- MySQL
- HTML / CSS / JavaScript
- Git / GitHub

## 工夫した点
入力が面倒だと継続して使えないと考え、最低限の情報でも登録できるようにしました。
また、同じ商品でも複数購入したい場合があるため、かぶりを検知しても自動で登録を中止せず、ユーザーが判断できるようにしています。
